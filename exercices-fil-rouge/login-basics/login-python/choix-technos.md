## Fast-API + SQLAlchemy

- Framework web moderne et rapide pour Python.
- ORM (Object-Relational Mapping) pour interagir avec la base de données.
- Bonne documentation et communauté active.


## Pourquoi pas Django

- Framework web complet, mais peut être trop lourd pour des applications simples.
- Moins flexible que Fast-API pour des besoins spécifiques.
- Structure de projet plus rigide.

## Pourquoi pas Flask

- Framework minimaliste, mais nécessite plus de configuration pour des fonctionnalités avancées.
- Moins performant que Fast-API pour des applications asynchrones.
- Moins de fonctionnalités intégrées par défaut.
